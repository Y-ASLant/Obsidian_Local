# 使用PyCharm IDE 训练

> [!NOTE] Clone repo and install requirements.txt in a **Python>=3.8.0** environment, including **PyTorch>=1.8**

```sh
git clone https://github.com/ultralytics/yolov5.git
```

```sh
cd yolov5
```

```sh
pip install -r requirements.txt
```
