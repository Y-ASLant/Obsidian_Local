# LED 闪烁
> [!tip] 假设 LED 在 A2

```cpp
#include <Arduino.h>
void setup() {
  // 初始化引脚为输出模式
  pinMode(2, OUTPUT); // 假设LED连接到GPIO2
}

void loop() {
  // 打开LED
  digitalWrite(2, LOW);
  // 等待一秒钟
  delay(1000);
  // 关闭LED
  digitalWrite(2, HIGH);
  // 再次等待一秒钟
  delay(1000);
}
```
