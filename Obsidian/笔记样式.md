# 插入视频
### 在线视频
```html
 <div style="text-align: center;"><video src="mp4" width="100%" height="100%" controls="controls" class="rounded-video"></video></div>
```

#### 在线视频样式
```css
.rounded-video {
    border-radius: 5px;
}
```
